![Demo](./screen.png)


## How to run
Clone the repository

Install dependencies
```
npm i
```

Check for error
```
truffle compile
```
Migrate
```
truffle migrate --reset
```
Run web
```
npm run start
```

You need to configure MetaMask before with your account, for test use ganache
